# Expense Analytics CLI

## Project Topic
A Python command-line tool that reads expense data from a CSV file and computes useful analytics.

## Core Features
- Load expenses from CSV
- Compute total spending
- Compute category-level totals
- Compute mean spending
- Compute median spending
- Compute standard deviation of spending
- Print a simple summary report

## Why This Topic
This project is small enough to complete cleanly, but still large enough to demonstrate version control workflows, AI-assisted code review, commit discipline, merge conflict handling, and CI/CD.

## Planned Tech Stack
- Python
- Git / GitHub
- GitHub Actions
- CSV input data
